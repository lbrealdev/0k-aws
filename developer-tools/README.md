# AWS Developer Tools

AWS Developer Tools is a set of services designed to help developers build, test, deploy, and manage applications on AWS.

## Services

| Service          | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| **CodeCommit**   | Fully-managed source control service that hosts Git repositories            |
| **CodeArtifact** | Fully-managed artifact repository compatible with npm, pip, Maven, and more |
| **CodeBuild**    | Fully-managed continuous integration service that compiles source code      |
| **CodeDeploy**   | Fully-managed deployment service that automates application deployments     |
| **CodePipeline** | Fully-managed continuous delivery service that automates release pipelines  |
